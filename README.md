# Own Enumerable

This project contains a class and a module in which we created our own enumerables which were then used in the class so that instances of the class can access the enumerables.

# Built with

- Ruby

# Setup Instructions

1. Clone this repository
2. Run `bundle install` in your terminal
3. Open `irb` terminal
4. Run `load 'main.rb'` to execute the code
